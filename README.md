# Ranking Pung Api module
This project is api module for ranking-pung service. 

- Realtime Rank
- News about rank-keyword
- Images about rank-keyword
- Anonymous board
- Message will be deleted over time

## Build & Run Server
```
./mvnw install
java -jar ./target/*.jar
```
